# Seguridad del Sistema

## Configuración Protegida
- Las claves de Firebase están protegidas mediante variables de entorno
- No se exponen credenciales en el código público
- Uso de reglas de seguridad en Firebase

## Reporting de Vulnerabilidades
Si encuentras una vulnerabilidad de seguridad, por favor contacta directamente al administrador.

## Mejores Prácticas
- Nunca committear archivos .env
- Rotar claves API regularmente
- Mantener dependencias actualizadas
